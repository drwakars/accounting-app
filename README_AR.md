# دفتر الحسابات - تطبيق محاسبة للآيفون

## المميزات

### 1. المصادقة وإدارة المستخدمين
- تسجيل دخول آمن مع JWT
- دعم صلاحيات (مدير/مستخدم)
- إدارة المستخدمين للمديرين

### 2. إدارة الحسابات
- إضافة أشخاص مع أرصدة ابتدائية (USD & IQD)
- البحث في الأشخاص
- عرض الأرصدة الحالية تلقائياً

### 3. العمليات المالية
- تسجيل إيداع وسحب
- دعم عملتين (دولار أمريكي ودينار عراقي)
- تعديل وحذف العمليات
- حماية من السحب الزائد

### 4. التقارير
- تقرير شهري شامل
- عرض الإيداعات والسحوبات لكل عملة
- حساب الصافي تلقائياً

### 5. سجل التدقيق
- تسجيل جميع العمليات
- معرفة من قام بأي عملية ومتى

## البيانات الافتراضية

## التقنيات المستخدمة

### Frontend:
- React Native (Expo)
- TypeScript
- Expo Router (للتنقل)
- Zustand (لإدارة الحالة)
- AsyncStorage (للتخزين المحلي)

### Backend:
- FastAPI (Python)
- MongoDB (قاعدة البيانات)
- JWT (للمصادقة)
- BCrypt (لتشفير كلمات المرور)

## كيفية بناء ملف IPA للآيفون

### الطريقة 1: استخدام EAS Build (موصى بها)

1. **تثبيت EAS CLI:**
\`\`\`bash
npm install -g eas-cli
\`\`\`

2. **تسجيل الدخول:**
\`\`\`bash
eas login
\`\`\`

3. **تهيئة المشروع:**
\`\`\`bash
cd /app/frontend
eas build:configure
\`\`\`

4. **بناء IPA للـ iOS:**
\`\`\`bash
eas build --platform ios --profile production
\`\`\`

5. **تحميل الملف:**
   - بعد اكتمال البناء، ستحصل على رابط لتحميل ملف IPA
   - يمكنك تثبيته باستخدام TestFlight أو مباشرة

### الطريقة 2: تجربة فورية بدون IPA

1. **حمّل تطبيق Expo Go على آيفونك:**
   - من App Store: [Expo Go](https://apps.apple.com/app/expo-go/id982107779)

2. **افتح التطبيق واختر "Scan QR Code"**

3. **امسح الـ QR Code الظاهر في Terminal أو الرابط:**
   - الرابط: https://offline-ios-app-2.preview.emergentagent.com

### الطريقة 3: بناء محلي (يتطلب Mac)

1. **تثبيت Xcode من App Store**

2. **تشغيل البناء:**
\`\`\`bash
cd /app/frontend
npx expo run:ios
\`\`\`
