# تطبيق دفتر الحسابات - Accounting Ledger App

## 📥 تحميل المشروع

### الطريقة 1: تحميل ZIP
اضغط على الزر الأخضر "Code" ثم "Download ZIP"

### الطريقة 2: Git Clone
```bash
git clone <repository-url>
```

---

## 🚀 البدء السريع

```bash
# 1. فك الضغط (إذا حملت ZIP)
cd accounting-ledger-app

# 2. ثبّت المكتبات
cd frontend
npm install -g yarn eas-cli
yarn install

# 3. سجل دخول وابني
eas login
eas build --platform ios --profile production
```
